# Task 1 - Logging Library

Node.js-sovellus joka demonstroi Winston-lokituskirjaston käyttöä.

## Käytetyt versiot
- Node.js: 18+
- Winston: 3.11.0
- Mocha: 10.x

## Asennus
npm install

## Käyttö
node src/main.js

## Testit
npm test

## Huomiot
- Timestamp-formaatti muutettu muotoon DD-MM-YYYY HH:mm:ss (paikallinen aika)
- Lokitiedostot (logs/) on jätetty pois git-historiasta .gitignore-tiedostolla
